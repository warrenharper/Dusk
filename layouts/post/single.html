Parallax Scrolling without javascript Yesterday I was wondering if it would be possible to create a parallax header, like the one used on Medium, using only CSS. I figured it should be possible using 3d transforms. Turns out Scott Kellum has already made a demo on CodePen using just that. I wanted to see it work on something else than images of kittens, so I made this little demo. Unfortunately this doesn't work well in Chrome on Retina screens (which is an issue that's being tracked) so use this with caution. Of course this won't work in older browsers, but you can pretty safely use it as an enhancement.
